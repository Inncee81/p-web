    // When the user scrolls down, hide the header.

    var prevScrollpos = window.pageYOffset,
        bodyEl = document.querySelector('body'),
        scrollDirection = null;

    function headerScroll() {
      try {
        var currentScrollPos = window.pageYOffset,
          scrollTop = window.pageYOffset || document.documentElement.scrollTop;
        if (scrollTop > 40) {
          if (Math.abs(prevScrollpos - currentScrollPos) > 40) {
            // Trigger only with previous direction was not up.
            if (prevScrollpos >= currentScrollPos) {
              if (scrollDirection !== 'up') {
                scrollDirection = 'up';
                bodyEl.classList.remove('js-scroll-dir--down');
                bodyEl.classList.remove('js-scroll-dir--top');
                bodyEl.classList.add('js-scroll-dir--up');
              }
            }
            // Trigger only with previous direction was not down.
            else {
              if (scrollDirection !== 'down') {
                scrollDirection = 'down';
                bodyEl.classList.remove('js-scroll-dir--up');
                bodyEl.classList.remove('js-scroll-dir--top');
                bodyEl.classList.add('js-scroll-dir--down');
              }
            }
          }
        }
        else {
          bodyEl.classList.remove('js-scroll-dir--top');
          bodyEl.classList.remove('js-scroll-dir--up');
        }
        prevScrollpos = currentScrollPos;
      } catch (error) {
        document.removeEventListener('scroll', headerScroll);
        console.log(error);
      }
    }
    document.addEventListener('scroll', headerScroll);
  })();

                <h2 class="feature-longforms__article-title heading__one no-space align-center">
                 <a data-ga-event-action="longform" data-ga-event-category="homepage" data-ga-event-label="/2421344/glanville-covid-19-cure" data-ga-event-on="click" href="/2421344/glanville-covid-19-cure">
                  The Antibody Avenger and the Quest for a COVID-19 Cure
                 </a>
                </h2>
                <div class="feature-longforms__article-dek dek align-center">
                 <p class="no-space">
                  Vaccines are rolling out with increasing speed, but we’ll also need effective treatments, because new coronavirus cases will be a worldwide reality for years to come. Enter Jacob Glanville, a maverick San Francisco immunologist who believes he’s found an unparalleled path to healing.
                 </p>
                </div>
                <div class="feature-longforms__article-body post-entry spacing" id="longform_ab_body">
                 <p>
                  Since the early days of the pandemic, Jacob Glanville, a 40-year-old skateboarding immunologist with a silver tongue and a knack for self-promotion, has been proclaiming that he would develop the world’s best COVID-19 treatment. Few competitors believed him. Most just scoffed. But Glanville, then the chief executive officer of a tiny San Francisco company called Distributed Bio, has made significant progress, successfully creating an antibody that neutralizes the coronavirus in hamsters, an animal model for humans. He’s far behind the pharmaceutical giants Regeneron and Eli Lilly—both of which
                  <strong>
                  </strong>
                  won
                  <a href="https://www.fda.gov/news-events/press-announcements/coronavirus-covid-19-update-fda-authorizes-monoclonal-antibody-treatment-covid-19">
                   FDA approval
                  </a>
                  of antibody-based treatments in November 2020—but his work is still important. With a total of $9 million invested so far, his discoveries have cost only a fraction of what the big firms have spent on their research. He says his treatment will work well and be more affordable—think $900 per dose instead of the $2,000 per that Regeneron charges. “I’ve got the best-in-class therapeutic,” Glanville told me recently, repeating a line he’s used often since April 2019.
                 </p>
                 <p>
                  Given the FDA’s approval of the first vaccine on December 11, 2020, you might be wondering why you should care about Glanville or his drug. It’s because
                  <a href="https://www.theatlantic.com/health/archive/2021/02/herd-immunity-might-be-impossible-even-vaccines/617973/">
                   vaccines alone
                  </a>
                  won’t end the pandemic. That’s especially true in the U.S., where so many people see their freedom to get sick and pass it along as a basic right. The chilling truth is that, at this point, it’s effectively impossible to vaccinate enough people worldwide to vanquish COVID-19. And so,
                  <strong>
                  </strong>
                  as with the flu, the deadly virus is likely to remain resident among us, destined to cause breakouts every year. You should care about Glanville—along with the many other drug manufacturers still pushing their creations forward—because, odds are, the best COVID-19 treatments aren’t available yet. If Glanville is right, he could save many lives all over the planet.
                 </p>
                </div>
