            <p>
             カーリースと目にすると、「レンタカーみたいなものでしょ？」と思いますよね。間違いでもなく、カーリースとは長期間乗り続けることができるレンタカーと言えば分かりやすいかもしれません。現在では、カーリースの分野においても法人向け・個人向けのカーリースが可能になっています。
             <br/>
             今回は、カーリースとは何か、基本的な種類やシステム、契約の流れについて順を追って見ていきましょう。
            </p>
            <p>
             ◆まず、カーリースとは何か？
             <br/>
             カーリースとは、月々一定の支払いで車を使えるシステムです。
             <br/>
             車を資産として所有しないことで、一時払いの負担や税金資産計上の必要もなく、車両を継続的に保有する場合に必要な、税金や代替などで発生する経費を予定する必要はありません。
            </p>
            <p>
             ◆カーリースの種類
             <br/>
             カーリースを大きく分けると2種類になります。リース会社によって、その名称やリース料金に含まれる内容に相違点があります。まずは、2つのカーリースの特徴を見てみましょう。
             <br/>
             ➀ファイナンスリース
             <br/>
             点検・整備・修理などは利用者の負担になりますが、車両代金及び自動車諸税・保険料などの費用をリース会社が負担し、利用者の調達ニーズに応えています。
             <br/>
             いわば、金融的性格が強いリース方式です。
             <br/>
             ➁メンテナンスリース
             <br/>
             車両の購入から、維持管理までのトータルサポートを受けることが出来ます。
             <br/>
             調達ニーズというファイナンスリースのメリットに加え、メンテナンスリースは車両のマネージメントサービスという付加価値を高めたシステムであり、最近ではメンテナンスリースが全体の60％を超え主流となっています。
            </p>
            <p>
             ◆カーリースのシステム
             <br/>
             カーリースでは、車の所有者はリース会社などのファイナンス会社になります。利用者はファイナンス会社が自動車メーカーやディーラーから購入した車を、利用期間やプランを決めて借り受けるシステムです。
             <br/>
             そのため、車検証の所有者欄には「ファイナンス会社名」、利用者欄には「ユーザー名」が記載されることになります。
             <br/>
             〇契約までの流れ
             <br/>
             一般的な契約では、以下の様な流れになります。
             <br/>
             ➀希望車種の決定　→　➁見積　→　➂契約申し込み及び内容確認　→　➃契約
             <br/>
             →　➄車両発注　→　➅ナンバー登録・納車・リース開始
             <br/>
             ◆カーリースの契約満了後はどうするのか？
             <br/>
             リース契約期間満了時は、以下の様な選択が出来ます
             <br/>
             ➀継続して利用する
             <br/>
             残価などを基にした再リース契約で継続して利用する
             <br/>
             ➁新しい車への乗換え
             <br/>
             残価を清算して新しい車へ乗り換える
             <br/>
             ➂購入する
             <br/>
             残価＋手数料および自動車リサイクル預託金相当額などを支払い購入する
             <br/>
             ➃返却する
             <br/>
             残価を清算して返却する
             <br/>
             ※残価とは、リース契約期間満了時の予想車両販売価格のことです
            </p>
            <p>
             ◆カーリースのメリット
             <br/>
             まず、一番のメリットは、自動車の維持についての負担が少ないことが挙げられます。
             <br/>
             自動車を所有すると、車検や保険など一時的に大きなお金が必要な場合がありますが、リースは月々の支払いが定額になるので、精神的にも経済的にもとても安心。カーリースはリース会社が車を大量購入する背景があるので、安く車両を借りることができるのです。
             <br/>
             利用者が払う月々の支払いはリース料のみなので、家計の予算は非常に立てやすいといえます。自動車の所有はリース会社となるため、車検やメンテナンスはすべてリース会社が行うことになり、面倒な手続きは一切必要なくなります。
            </p>
            <p>
             ◆カーリースの注意点
             <br/>
             カーリースを利用する場合は、いくつか注意点があります。
             <br/>
             ➀利用には、申し込み後に与信審査があります。
             <br/>
             利用者が申込後、各リース会社が与信審査を実施します。与信審査が通ってはじめてカーリースを利用することができます。
             <br/>
             ➁リース期間中は途中解約できません。
             <br/>
             基本的に、リース契約期間中の中途解約は出来ません。
            </p>
            <p>
             カーリースとは、レンタカーを長期間借りているようなものであり、月々の支払いも定額で面倒な手続きや支払などがありません。車を色々と乗り換えてみたい方や、税金など煩わしい手続きは嫌という方には勿論、個人事業主や法人の方に特にオススメです。
            </p>
