       <a class="fn db f16 ellipsis mb5" href="//iask.fh21.com.cn/question/105454072.html" target="_blank">
        汗管瘤有什么治疗方法
       </a>
       <a class="hidden2 db f14 fc6 wbwr des" href="//iask.fh21.com.cn/question/105454072.html" target="_blank">
        回答：汗管瘤大多是通过物理治疗，如激光、冷冻、酸蚀等，但这些方法只是暂时切除，一般过几个月后就会复发，有时比原来长得还要多，而且容易形成疤痕、色素沉着。根据汗管瘤的病因，可采用激光结合中药治疗，内外兼治，调节机体免疫力，配合活血化瘀药外敷，清热解毒，散结消肿，从根本上解决问题，一般治愈后不会在复发。建议平时饮食清淡，多吃新鲜水果和蔬菜，避免油炸和辛辣刺激食物等。
       </a>
      </div>

       <a class="fn db f16 ellipsis mb5" href="//iask.fh21.com.cn/question/105454018.html" target="_blank">
        肾移植手术后多久尿量正常
       </a>
       <a class="hidden2 db f14 fc6 wbwr des" href="//iask.fh21.com.cn/question/105454018.html" target="_blank">
        回答：做完肾移植手术之后，也不代表尿量越多越好，正常人的尿量大概在1500毫升左右，术后短暂的排尿是由于肾小管浓缩功能正在恢复，所以尿量会有所减少，大概做完肾移植手术之后，3~5天的时候就可以恢复正常的尿量，一般术后肾功能恢复正常，尿量就不会出现异常，但是尽量不要故意喝更多的水，因为喝水过多还可能会给肾脏增加负担，做完肾移植手术之后，后期也需要到医院进行复查。
       </a>

       <a class="fn db f16 ellipsis mb5" href="//iask.fh21.com.cn/question/105454035.html" target="_blank">
        大腿内侧有红色条纹是什么原因
       </a>
       <a class="hidden2 db f14 fc6 wbwr des" href="//iask.fh21.com.cn/question/105454035.html" target="_blank">
        回答：通常是由于生长纹造成的。或者是由于休重短时间提升过快，也可能是生长发育的速率过快，造成肌肤化学纤维破裂而产生的。通常情况下生长纹不是危害生长发育和生长发育的，但会危害美观大方。能够外敷维生素药膏相互配合尿素乳膏涂擦改进病症，可是药品只可以具有必须的减轻功效，是没法彻底消散的。中后期生长纹会色调变浅衰老，可以考虑使用激光手术进行治疗。
       </a>

function dateFtt(fmt,date)   { 
  date = new Date(date*1000); 
  var o = {   
    "M+" : date.getMonth()+1,                 //月份
    "d+" : date.getDate(),                    //日
    "h+" : date.getHours(),                   //小时
    "m+" : date.getMinutes(),                 //分
    "s+" : date.getSeconds(),                 //秒
    "q+" : Math.floor((date.getMonth()+3)/3), //季度
    "S"  : date.getMilliseconds()             //毫秒
  };   
  if(/(y+)/.test(fmt)){
      fmt=fmt.replace(RegExp.$1, (date.getFullYear()+"").substr(4 - RegExp.$1.length));
      for(var k in o){
        if(new RegExp("("+ k +")").test(fmt)){
            fmt = fmt.replace(RegExp.$1, (RegExp.$1.length==1) ? (o[k]) : (("00"+ o[k]).substr((""+ o[k]).length))); 
        }
      }
  }
  return fmt;
}
