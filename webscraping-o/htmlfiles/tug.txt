    <p>
     <b>
      The TeX Users Group (TUG)
     </b>
     is a membership-based not-for-profit
organization, founded in 1980, for anyone who uses the TeX typesetting
system created by
     <a href="https://www-cs-faculty.stanford.edu/~knuth/">
      Donald Knuth
     </a>
     and/or is interested in typography and font design.
    </p>
    <p>
     <b>
      Please consider
      <a href="https://tug.org/forms/current/memberapp.html">
       joining TUG
      </a>
     </b>
     (
     <a href="https://tug.org/forms/current/trialapp.html">
      trial
memberships
     </a>
     available for new members)
to support use and development of TeX and friends.
     <a href="https://lists.tug.org/tex-announce">
      Subscribe to our
monthly newsletter
     </a>
     (tex-announce) if you like (automatically sent to
all members).
    </p>
    <p id="new">
     <b>
      If you're new to TeX
     </b>
     and are looking for a system to
install, or have questions to ask,
     <a href="/begin.html">
      click here
to get started
     </a>
     .
    </p>
    <p id="help">
     <b>
      If you're looking for help
     </b>
     , you can get community
support via: a
     <a href="https://tex.stackexchange.com/" rel="nofollow">
      q&amp;a site
     </a>
     (tex.stackexchange.com),
     <a href="https://latex.org/forum/">
      forum
     </a>
     (latex.org),
     <a href="https://www.reddit.com/r/LaTeX/">
      reddit
     </a>
     ,
     <a href="https://lists.tug.org/texhax">
      public mailing list
     </a>
     (texhax),
and
     <a href="/interest.html#doc">
      in other ways
     </a>
     .

    <b>
     The
     <a href="https://ctan.org/">
      Comprehensive TeX Archive Network
     </a>
     (CTAN)

     topic cloud
    </a>
    , and
    <a href="https://ctan.org/search.html">
     CTAN search page
    </a>
    can help you
find what you need.
