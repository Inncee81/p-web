     </p>
     <p>
      <font face="arial" size="2">
       <b>
        Features in
       </b>
      </font>
      <font color="#FF0000" face="Arial" size="2">
       <b>
        Total Commander
        9.51
       </b>
      </font>
      <font face="arial" size="2">
       <b>
        now
        include :
       </b>
      </font>
     </p>
     <ul type="disc">
      <li>
       <font face="arial" size="2">
        Two file windows side
                by side
       </font>
      </li>
      <li>
       <font face="arial" size="2">
        Multiple language and
                Unicode support
       </font>
      </li>
      <li>
       <font face="arial" size="2">
        Enhanced search
                function
       </font>
      </li>
      <li>
       <font face="arial" size="2">
        Compare files (now
                with editor) / synchronize directories
       </font>
      </li>
      <li>
       <font face="arial" size="2">
        Quick View panel with
                bitmap display
       </font>
      </li>
      <li>
       <font face="arial" size="2">
        ZIP, 7ZIP, ARJ, LZH,
                RAR, UC2, TAR, GZ, CAB, ACE archive handling +
                plugins
       </font>
      </li>
      <li>
       <font face="arial" size="2">
        Built-in FTP client
                with FXP (server to server) and HTTP proxy
                support
       </font>
      </li>
      <li>
       <font face="arial" size="2">
        Parallel port link,
                multi-rename tool
       </font>
      </li>
      <li>
       <font face="arial" size="2">
        Tabbed interface,
                regular expressions, history+favorites buttons
       </font>
      </li>
      <li>
       <font face="arial" size="2">
        Thumbnails view,
                custom columns
       </font>
      </li>
      <li>
       <font face="arial" size="2">
        Compare editor,
                cursor in lister, separate trees, logging,
                enhanced overwrite dialog etc.
       </font>
      </li>
      <li>
       <font face="arial" size="2">
        Unicode names almost
                everywhere, long names (&gt;259 characters),
                password manager for ftp and plugins, synchronize
                empty dirs, 64 bit context menu, quick file
                filter (Ctrl+S)
       </font>
      </li>
      <li>
       <font face="arial" size="2">
        USB port connection
                via special
       </font>
       <a href="cables/index.htm">
        <font face="arial" size="2">
         direct transfer cable
        </font>
       </a>
       <font face="arial" size="2">
        , partial branch view
                (Ctrl+Shift+B), and many improvements to ftp,
                synchronizing and other functions
       </font>
      </li>
      <li>
       <a href="featurel.htm">
        <font face="arial" size="2">
         And many more!
        </font>
       </a>
      </li>
     </ul>
