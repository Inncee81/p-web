      <h2>
       <span class="mw-headline" id="WMCS_project_overview">
        WMCS project overview
       </span>
      </h2>
      <h3>
       <span id="What_is_Wikimedia_Cloud_Services_.28WMCS.29.3F">
       </span>
       <span class="mw-headline" id="What_is_Wikimedia_Cloud_Services_(WMCS)?">
        What is Wikimedia Cloud Services (WMCS)?
       </span>
      </h3>
      <p>
       <b>
        Wikimedia Cloud Services
       </b>
       (
       <b>
        WMCS
       </b>
       ) provides tools, services, and support for technical collaborators who want to contribute to Wikimedia software projects. WMCS is a computing ecosystem built on
       <a class="extiw" href="https://en.wikipedia.org/wiki/OpenStack" title="w:OpenStack">
        OpenStack
       </a>
       ,
       <a class="extiw" href="https://en.wikipedia.org/wiki/Oracle_Grid_Engine" title="w:Oracle Grid Engine">
        GridEngine
       </a>
       , and
       <a class="extiw" href="https://en.wikipedia.org/wiki/Kubernetes" title="w:Kubernetes">
        Kubernetes
       </a>
       .
      </p>
      <p>
       WMCS products and services are available for use by anyone connected with the
       <a class="extiw" href="https://meta.wikimedia.org/wiki/Wikimedia_movement" title="meta:Wikimedia movement">
        Wikimedia movement
       </a>
       without charge. Support and administration of the WMCS resources is provided by a
       <a class="extiw" href="https://www.mediawiki.org/wiki/Wikimedia_Cloud_Services_team" title="mw:Wikimedia Cloud Services team">
        Wikimedia Foundation Cloud Services team
       </a>
       and
       <a class="extiw" href="https://foundation.wikimedia.org/wiki/Volunteer_opportunities" title="wmf:Volunteer opportunities">
        Wikimedia movement volunteers
       </a>
       . We maintain a
       <a href="/wiki/Help:Glossary" title="Help:Glossary">
        Glossary
       </a>
       of related terminology.
      </p>
      <p>
       🎬
       <b>
        Video
       </b>
       :
       <a class="external text" href="https://media.ccc.de/v/36c3-77-wikimedia-cloud-services-introduction" rel="nofollow">
        Wikimedia Cloud Services introduction
       </a>
      </p>
      <p>
       📣
       <b>
        Slides
       </b>
       :
       <a class="extiw" href="https://commons.wikimedia.org/wiki/File:Introduction_to_Wikimedia_Cloud_Services_-_Wikimania_Hackathon_2019_Stockholm_Sweden.pdf" title="commons:File:Introduction to Wikimedia Cloud Services - Wikimania Hackathon 2019 Stockholm Sweden.pdf">
        An introduction to Cloud Services presentation from Wikimania Stockholm 2019
       </a>
      </p>
      <h3>
       <span class="mw-headline" id="WMCS_history">
        WMCS history
       </span>
      </h3>
      <p>
       From 2011 until early 2017, WMCS was known as "Wikimedia Labs." The term 'Labs' was used to refer to a number of different
       <a class="mw-redirect" href="/wiki/Labs_labs_labs" title="Labs labs labs">
        components
       </a>
       , and clarification was required. In 2017, the project was reorganized. The former Wikimedia Foundation Labs team and the Tool Labs Support team joined together to create the Wikimedia Cloud Services team.
      </p>

       <span class="mw-headline" id="Renaming_of_products_and_services">
        Renaming of products and services
       </span>
      </h3>
      <p>
       We are in the process of changing the
       <a class="extiw" href="https://phabricator.wikimedia.org/phame/post/view/59/labs_and_tool_labs_being_renamed/" title="phab:phame/post/view/59/labs and tool labs being renamed/">

       </a>
       of the products and services we offer. You may find some outdated titles and names in WMCS documentation. Edits are welcome!
      </p>
      <h2>
       <span class="mw-headline" id="Participating_with_WMCS">
        Participating with WMCS
       </span>
      </h2>
      <h3>
       <span class="mw-headline" id="Sign_up_for_services">
        Sign up for services
       </span>
      </h3>
      <p>
       To access and contribute to Cloud Services projects and tools, you will need the following accounts:
      </p>
      <ul>
       <li>
        <b>
         Wikimedia account
        </b>
        - this account is the single user login or SUL account you use to contribute to Wikipedia and its sister projects.
       </li>
       <li>
        <a href="/wiki/Help:Create_a_Wikimedia_developer_account" title="Help:Create a Wikimedia developer account">
         <b>
          Wikimedia developer account
         </b>
        </a>
        - this account is used log into this wiki, Toolforge, Cloud VPS, Gerrit and other protected Wikimedia Services.
       </li>
       <li>
        <b>
         <a class="extiw" href="https://www.mediawiki.org/wiki/Gerrit" title="mw:Gerrit">
          Gerrit
         </a>
        </b>
        - our code review system; where our repositories (repos) live. Note that while
        <a class="extiw" href="https://www.mediawiki.org/wiki/Github" title="mw:Github">
         Github
        </a>
        contains many of our public repos, you can only make pull requests for Cloud Services projects via Gerrit. Other Wiki projects may use Github exclusively.
       </li>
       <li>
        <b>
         <a class="extiw" href="https://www.mediawiki.org/wiki/Phabricator" title="mw:Phabricator">
          Phabricator
         </a>
        </b>
        - our project management system; for opening tickets, suggesting features, and talking about our plans for the next quarter.
       </li>
       <li>
        <b>
         <a class="extiw" href="https://www.mediawiki.org/wiki/MediaWiki_on_IRC" title="mw:MediaWiki on IRC">
          IRC
         </a>
        </b>
        - live chat channels. We have several channels related to our cloud servers, but the main channel is #wikimedia-cloud. Deployment also frequently uses #wikimedia-serviceops.
       </li>
       <li>
        The ability to
        <a class="mw-redirect" href="/wiki/Help:Access" title="Help:Access">
         access instances
        </a>
        in the WMCS environment.
       </li>
      </ul>
      <h3>
       <span class="mw-headline" id="Review_the_terms_and_conditions">
        Review the terms and conditions
       </span>
      </h3>
      <p>
       Second, make sure to review and agree to our terms and conditions.
       <a class="mw-redirect" href="/wiki/Help:Terminology" title="Help:Terminology">
        Account holders
       </a>
       who plan to use WMCS resources and products must read and agree to the following:
      </p>
      <ul>
       <li>
        <a href="/wiki/Wikitech:Cloud_Services_Terms_of_use" title="Wikitech:Cloud Services Terms of use">
         Wikimedia Cloud Services Terms of Use
        </a>
       </li>
       <li>
        <a class="extiw" href="https://www.mediawiki.org/wiki/Code_of_Conduct" title="mw:Code of Conduct">
         Code of Conduct for technical spaces
        </a>
       </li>
       <li>
        <a class="extiw" href="https://www.mediawiki.org/wiki/Wikimedia_Labs/Agreement_to_disclosure_of_personally_identifiable_information" title="mw:Wikimedia Labs/Agreement to disclosure of personally identifiable information">
         Agreement to disclosure of personally identifiable information
        </a>
        (covers
        <a class="mw-redirect" href="/wiki/Help:Terminology" title="Help:Terminology">
         End-Users
        </a>
        ).
       </li>
      </ul>
      <p>
       Please pay close attention to the following terms for
       <b>
        Toolforge and Cloud VPS:
       </b>
      </p>
      <ul>
       <li>
        Toolforge tools must be
        <a class="extiw" href="https://en.wikipedia.org/wiki/Open-source_software" title="w:Open-source software">
         open source software
        </a>
        licensed under an
        <a class="external text" href="http://opensource.org/licenses" rel="nofollow">
         OSI approved license
        </a>
        .
       </li>
       <li>
        Toolforge and Cloud VPS projects must not collect, store, or share private data or personally identifiable information, such as user names, passwords, or IP addresses, except when complying with the conditions listed in the
        <a href="/wiki/Wikitech:Cloud_Services_Terms_of_use" title="Wikitech:Cloud Services Terms of use">
         Wikimedia Cloud Services Terms of Use
        </a>
        .
       </li>
      </ul>
