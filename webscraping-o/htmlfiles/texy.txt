     <h1>
      What is Texy
     </h1>
     <div class="wiki-columns">
      <div class="wiki">
       <p>
        Texy allows you to enter content using an
        <b>
         easy to read
        </b>
        <a href="/en/syntax">
         Texy syntax
        </a>
        which is filtered into
        <em>
         structurally valid
        </em>
        HTML. No knowledge of HTML is required. Texy is one of the most complex formatting tools. It allows
adding of images, links, nested lists, tables and has full support for CSS.
       </p>
       <p>
        Texy supports hyphenation of long words (which reflects language rules), clickable emails and URL (emails are obfuscated
against spambots), national typographic single and double quotation marks, ellipses, em dashes, dimension sign, nonbreakable
spaces (e.g. in phone numbers), acronyms, arrows and many others.
       </p>
       <p>
        Texy is being developed by David Grudl since 2004. It is written in PHP.
       </p>
       <p>
        Texy is licenced by BSD and GNU General Public License. Plugins for several content-management systems are available.
       </p>
       <h1 id="toc-features">
        Features
       </h1>
       <ul>
        <li>
         simple, intuitive and human friendy markup
        </li>
        <li>
         generate clean and valid HTML code
        </li>
        <li>
         may be used together with syntax highlighter
        </li>
        <li>
         designed with proper typography in mind
        </li>
        <li>
         supports hyphenation
        </li>
       </ul>
       <h1 id="toc-why-texy">
        Why Texy
       </h1>
       <ul>
        <li>
         bulletproof – ensures the well-formedness of the resulting code
        </li>
        <li>
         predectable behaviour
        </li>
        <li>
         flexible configuration
        </li>
       </ul>
