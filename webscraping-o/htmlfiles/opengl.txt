      <h2>
       <a href="https://developer.nvidia.com/designworks">
        NVIDIA provides OpenGL-accelerated Remote Desktop for GeForce
       </a>
      </h2>
      <p>
       In these days of social distancing, game developers and content creators all over the world are working from home and asking for help using Windows Remote Desktop streaming with the OpenGL tools they use. NVIDIA has created a special tool for GeForce GPUs to accelerate Windows Remote Desktop streaming with
       <a href="https://www.nvidia.com/drivers">
        GeForce drivers
       </a>
       R440 or later. Download and run the
       <a href="https://developer.nvidia.com/nvidia-opengl-rdp" target="_blank">
        executable
       </a>
       (nvidiaopenglrdp.exe) from the DesignWorks website as Administrator on the remote Windows PC where your OpenGL application will run. A dialog will confirm that OpenGL acceleration is enabled for Remote Desktop and if a reboot is required.
      </p>

      <h2>
       <a href="https://www.khronos.org/news/press/khronos-group-releases-vulkan-1.2">
        Khronos Group Releases Vulkan 1.2
       </a>
      </h2>
      <p>
       The Khronos Group
       <a data-href="https://www.khronos.org/news/press/khronos-group-releases-vulkan-1.2" href="https://www.khronos.org/news/press/khronos-group-releases-vulkan-1.2" title="https://www.khronos.org/news/press/khronos-group-releases-vulkan-1.2">
        announces the release of the Vulkan 1.2 specification
       </a>
       for GPU acceleration. This release integrates 23 proven extensions into the core Vulkan API, bringing significant developer-requested access to new hardware functionality, improved application performance, and enhanced API usability. Multiple GPU vendors have certified conformant implementations, and significant open source tooling is expected during January 2020. Vulkan continues to evolve by listening to developer needs, shipping new functionality as extensions, and then consolidating extensions that receive positive developer feedback into a unified core API specification. Khronos and the Vulkan community will support Vulkan 1.2 in a wide range of open source compilers, tools, and debuggers by the end of January 2020. Driver release updates will be posted on the
       <a href="https://github.com/KhronosGroup/Vulkan-Docs/issues/1162" target="_blank">
        Vulkan Public Release Tracker
       </a>
       .
      </p>
      <p>
       Find more information on the Vulkan 1.2 specification and associated tests and tools at:
      </p>
      <ul>
       <li>
        Khronos’
        <a data-href="https://www.khronos.org/vulkan/" href="https://www.khronos.org/vulkan/" title="https://www.khronos.org/vulkan/">
         Vulkan Resource Page
        </a>
       </li>
       <li>
        Sample code can be found in the Vulkan Unified Samples Repository
       </li>

       </li>

      <h2>
       <a href="https://news.developer.nvidia.com/nvidia-announces-nsight-systems-2019-6/">
        NVIDIA announces Nsight Systems 2019.6 with OpenGL improvements
       </a>
      </h2>
      <p>
       NVIDIA Nsight Systems 2019.6 is now available for download. This release expands graphics trace on Windows by adding support for Direct3D 11, WDDM CPU+GPU queues, and OpenGL. On Linux, new features include support for CUDA 10.2, simultaneous CLI sessions, DWARF unwind and capture by hotkey.
      </p>
