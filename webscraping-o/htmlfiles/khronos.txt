         <a href="https://www.phoronix.com/scan.php?page=news_item&amp;px=GRVK-0.3-Mantle-Vulkan" itemprop="url">
          GRVK 0.3 Released For Continuing To Implement AMD’s Deprecated Mantle API Atop Vulkan
         </a>
        </h3>
        <div class="articleMetadata">
         <span>
          <a aria-label="Go to the full news article about GRVK 0.3 Released For Continuing To Implement AMD’s Deprecated Mantle API Atop Vulkan" href="/news/permalink/grvk-0.3-released-for-continuing-to-implement-amds-deprecated-mantle-api-atop-vulkan-6053058eef01f3.99453626" itemprop="url">
           <i class="fa fa-link">
           </i>
          </a>
         </span>
         <time datetime="2021-03-18T00:46:00-07:00" itemprop="dateCreated">
          <i class="fa fa-clock-o">
          </i>
          March 18, 2021
         </time>
         <span class="articleTags" itemprop="keywords">
          <i class="fa fa-tags">
          </i>
          <a href="/news/tags/tag/vulkan" title="vulkan">
           vulkan
          </a>
         </span>
         <span>
          <i class="fa fa-building">
          </i>
          <a aria-label="Go to the full news article about AMD" href="/news/members/amd" itemprop="url">
           AMD
          </a>
         </span>
        </div>
        <div itemprop="articleBody">
         <img alt="GRVK 0.3 Released For Continuing To Implement AMD’s Deprecated Mantle API Atop Vulkan" src="/assets/uploads/ceimg/made/assets/uploads/news/news_graphics/Vulkan_500px_Dec16_200_77.png"/>
         <p>
          GRVK is the open-source project implementing AMD’s Mantle API on Vulkan. Mantle was the precursor to the Vulkan industry standard and while it’s no longer in use by AMD, there still are some games out there that allow making use of it.
         </p>
         <p>
          <a href="https://github.com/libcg/grvk/releases/tag/0.3.0">
           GRVK
          </a>
          allows for such Mantle games to run atop Vulkan not only for AMD drivers/GPUs but also with NVIDIA and Intel graphics too. The original Vulkan API is derived from and built off Mantle so GRVK is able to map rather nicely and has been making progress in recent months.
         </p>
        </div>
