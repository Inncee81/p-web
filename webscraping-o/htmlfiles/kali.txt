  <section id="kali-highlights">
   <div>
    <div>
     <h1>
      The Industry Standard
     </h1>
    </div>
   </div>
   <h3>
    Kali Linux is not about its tools, nor the operating system. Kali Linux is a
    <strong>
     platform
    </strong>
    .
   </h3>
   <ul>
    <li>
     <h4>
      <a href="/features/">
       <i class="ti-settings">
       </i>
      </a>
      Make Your Job Easier
     </h4>
     <p>
      You can take any Linux and install pentesting tools on it, but you have to set the tools up manually and configure them. Kali is optimized to reduce the amount of work, so a
      <a href="/features/">
       professional
      </a>
      can just sit down and go.
     </p>
    </li>
    <li>
     <h4>
      <a href="/downloads/">
       <i class="ti-target">
       </i>
      </a>
      Kali Everywhere
     </h4>
     <p>
      A version of Kali is always close to you, no matter where you need it. Mobile devices, Docker, ARM, Amazon Web Services, Windows Subsystem for Linux, Virtual Machine, bare metal, and others are all
      <a href="/downloads/">
       available
      </a>
      .
     </p>
    </li>
    <li>
     <h4>
      <i class="ti-pencil-alt">
      </i>
      Customization
     </h4>
     <p>
      With the use of
      <a href="/docs/general-use/metapackages/">
       metapackages
      </a>
      , optimized for the specific tasks of a security professional, and a highly accessible and well documented
      <a href="/docs/development/live-build-a-custom-kali-iso/">
       ISO customization process
      </a>
      , it's always easy to generate an optimized version of Kali for your specific needs.
     </p>
    </li>
    <li>
     <h4>
      <a href="/docs/">
       <i class="ti-book">
       </i>
      </a>
      Documentation
     </h4>
     <p>
      Whether you are a seasoned veteran or a novice, our
      <a href="/docs/">
       documentation
      </a>
      will have all the information you will need to know about Kali Linux. Multiple tips and “recipes” are available, to help ease doubts or address any issues. All documentation is open, so you can easily contribute.
     </p>
    </li>
    <li>
     <h4>
      <a href="/community/">
       <i class="ti-comments">
       </i>
      </a>
      Community
     </h4>
     <p>
      Kali Linux, with its
      <a href="https://www.backtrack-linux.org/" rel="noopener" target="_blank">
       BackTrack
      </a>
      lineage, has a vibrant and
      <a href="/community/">
       active community
      </a>
      . There are active Kali forums, IRC Channel, Kali Tools listings, an open bug tracker system, and even community provided tool suggestions.
     </p>
    </li>
   </ul>

     <h5>
      Undercover Mode
     </h5>
     <p>
      Using Kali in an environment where you don't want to draw attention to yourself?
      <a href="/docs/introduction/kali-undercover/">
       Kali Undercover
      </a>
      is the perfect way to not stand out in a crowd.
     </p>

     <h5>
      Kali NetHunter
     </h5>
     <p>
      A mobile penetration testing platform for Android devices, based on Kali Linux.
      <a href="/kali-nethunter/">
       Kali NetHunter
      </a>
      is made up of an App, App Store, Kali Container and KeX
     </p>

     <figure class="__js-animation" data-animation-duration="10000" id="winkex-demo">

     <h5>
      Win-KeX
     </h5>
     <p>
      <a href="/docs/wsl/win-kex/">
       Win-KeX
      </a>
      provides a full Kali Desktop Experience for Windows WSL. Applications started via Kali's panel will share the desktop with Microsoft Windows applications.
     </p>
    </div>
   </div>
   <ul>
    <li>
     <a href="/docs/arm/">
      <img alt="ARM architecture icon" src="images/kali-everywhere-arm.svg"/>
      <h3>
       ARM
      </h3>
     </a>
     <div>
      Cheap &amp; low powered devices, which make for great leave behind devices. Modern ARM based laptops provide high speed with long battery life as an assessment platform.
     </div>
    </li>
    <li>
     <a href="/docs/installation/">
      <img alt="Bare Metal installation icon" src="images/kali-everywhere-baremetal.svg"/>
      <h3>
       Bare Metal
      </h3>
     </a>
     <div>
      Single or multiple boot Kali, giving you complete control over the hardware access (perfect for in-built WiFi and GPU), enabling the best performance.
     </div>
    </li>
    <li>
     <a href="/docs/cloud/">
      <img alt="Cloud computing icon" src="images/kali-everywhere-cloud.svg"/>
      <h3>
       Cloud
      </h3>
     </a>
     <div>
      Hosting providers which have Kali Linux pre-installed, ready to go, without worrying about looking after the infrastructure.
     </div>
    </li>
    <li>
     <a href="/docs/containers/">
      <img alt="Container icon" src="images/kali-everywhere-container.svg"/>
      <h3>
       Containers
      </h3>
     </a>
     <div>
      Using Docker or LXD, allows for extremely quick and easy access to all of Kali's tools, without the overhead of a isolated virtual machine.
     </div>
    </li>
    <li>
     <a href="/docs/usb/">
      <img alt="Live USB icon" src="images/kali-everywhere-usb.svg"/>
      <h3>
       USB
      </h3>
     </a>
     <div>
      Kali in your pocket, ready to go with Live Boot. Your Kali, always with you, without altering the host OS, plus allows you to benefit from hardware access.
     </div>
    </li>
    <li>
     <a href="/docs/virtualization/">
      <img alt="Virtual Machine icon" src="images/kali-everywhere-vm.svg"/>
      <h3>
       Virtual Machines
      </h3>
     </a>
     <div>
      VMware &amp; VirtualBox pre-built images ready to go. Allowing for a Kali install as bare as possible, with additional features such as snapshots, without altering the host OS. And we have vagrant images too.
     </div>
    </li>
    <li>
     <a href="/docs/wsl/">
      <img alt="Windows Subsystem for Linux icon" src="images/kali-everywhere-wsl.svg"/>
      <h3>
       WSL
      </h3>
     </a>
     <div>
      WSL is included out of the box with modern Windows. You can then start to use Kali (and Win-Kex) without any installing any extra software.
     </div>
    </li>
   </ul>

  <section class="white-section" id="desktops-preview">
   <h1>
    Choose the desktop you prefer
   </h1>
   <div>
    <div>
     <h3>
      <img alt="Xfce logo" src="images/logo-xfce.svg"/>
      Xfce
     </h3>
     <p>
      Xfce is a lightweight desktop environment for UNIX-like operating systems. It aims to be fast and low on system resources, while still being visually appealing and user friendly.
     </p>
     <p>
      Xfce consists of separately packaged parts that together provide all functions of the desktop environment, but can be selected in subsets to suit user needs and preferences. This is Kali's default desktop environment.
     </p>
    </div>
    <img alt="Kali Linux with Xfce desktop" src="images/kali-desktop-xfce.jpg"/>
   </div>
   <div>
    <img alt="Kali Linux with GNOME desktop" src="images/kali-desktop-gnome.jpg"/>
    <div>
     <h3>
      <img alt="GNOME logo" src="images/logo-gnome.svg"/>
      GNOME Shell
     </h3>
     <p>
      Every part of GNOME 3 has been designed to make it simple and easy to use. The Activities Overview is an easy way to access all your basic tasks. A press of a button is all it takes to view your open windows, launch applications, or check if you have new messages. Having everything in one place is convenient and means that you don't have to learn your way through a maze of different technologies.
     </p>
    </div>
   </div>
   <div>
    <div>
     <h3>
      <img alt="KDE logo" src="images/logo-kde.svg"/>
      KDE Plasma
     </h3>
     <p>
      Plasma is made to stay out of the way as it helps you get things done. But under its light and intuitive surface, it's a powerhouse. So you're free to choose ways of usage right as you need them and when you need them.
     </p>
     <p>
      With Plasma the user is king. Not happy with the color scheme? Change it! Want to have your panel on the left edge of the screen? Move it! Don't like the font? Use a different one! Download custom widgets in one click and add them to your desktop or panel.
     </p>
    </div>
    <img alt="Kali Linux with KDE Plasma desktop" src="images/kali-desktop-kde.jpg"/>
   </div>

     <h4>
      Kali Linux 2021.1 Release (Command-Not-Found)
     </h4>
     <p>
      Today we’re pushing out the first Kali Linux release of the year with Kali Linux 2021.1. This edition brings enhancements of existing features, and is ready to be downloaded or upgraded if you have an existing Kali Linux installation.
The summary of the changelog since the 2020.4 release from November 2020 is:

    <a class="card" href="https://www.kali.org/blog/empire-starkiller/">
     <div style="background-image:url(https://www.kali.org/blog/empire-starkiller/images/kali-bc-security-partnership.png)">

     <h4>
      BC Security's Empire/Starkiller &amp; Kali Linux
     </h4>
