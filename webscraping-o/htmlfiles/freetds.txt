    <td>
     <p>
      <font size="+3">
       <b>
        F
       </b>
      </font>
      reeTDS is a set of libraries for Unix and Linux that allows your programs to natively talk to Microsoft SQL Server and Sybase databases.
     </p>
     <p>
      <font size="+3">
       <b>
        T
       </b>
      </font>
      echnically speaking, FreeTDS is an open source implementation of the TDS (Tabular Data Stream) protocol used by these databases for their own clients.  It supports many different flavors of the protocol and three APIs to access it.  Additionally FreeTDS works with other software such as Perl and PHP, providing access from those languages as well.
     </p>
     <p>
      If you are looking for a Java implementation, we refer you to the
      <a href="http://jtds.sourceforge.net/">
       jTDS project
      </a>
      on SourceForge.
     </p>
     <p>
      <font size="+3">
       <b>
        F
       </b>
      </font>
      reeTDS has many possible uses.  It has been used by Unix/Linux webservers to present data stored in SQL Server to the web, to port SQL Server database code from NT to Unix, to import data into SQL Server from a Unix source, and to provide database access on platforms (such as realtime systems) that have no native drivers.
     </p>
     <p>
      <font size="+3">
       <b>
        T
       </b>
      </font>
      he FreeTDS C libraries are available under the terms of the GNU LGPL license.  Consult COPYING.LIB in the distribution for details.
     </p>
