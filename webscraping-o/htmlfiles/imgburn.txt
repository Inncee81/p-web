             What is ImgBurn?
            </span>

           <td class="contents" colspan="2">
            <p>
             ImgBurn is a lightweight CD / DVD / HD DVD / Blu-ray burning application that everyone should have in their toolkit!
            </p>
            <p>
             It has several 'Modes', each one for performing a different task:
            </p>
            <ul>
             <li>
              <a href="index.php?act=screenshots#isoread" title="Click here to see a screenshot of this mode">
               <b>
                Read
               </b>
              </a>
              - Read a disc to an image file
             </li>
             <li>
              <a href="index.php?act=screenshots#isobuild" title="Click here to see a screenshot of this mode">
               <b>
                Build
               </b>
              </a>
              - Create an image file from files on your computer or network - or you can write the files directly to a disc
             </li>
             <li>
              <a href="index.php?act=screenshots#isowrite" title="Click here to see a screenshot of this mode">
               <b>
                Write
               </b>
              </a>
              - Write an image file to a disc
             </li>
             <li>
              <a href="index.php?act=screenshots#isoverify" title="Click here to see a screenshot of this mode">
               <b>
                Verify
               </b>
              </a>
              - Check a disc is 100% readable. Optionally, you can also have ImgBurn compare it against a given image file to ensure the actual data is correct
             </li>
             <li>
              <a href="index.php?act=screenshots#discovery" title="Click here to see a screenshot of this mode">
               <b>
                Discovery
               </b>
              </a>
              - Put your drive / media to the test! Used in combination with
              <a href="http://www.dvdinfopro.com/" target="_blank" title="Click here to visit the DVDInfoPro website (www.dvdinfopro.com)">
               <b>
                DVDInfoPro
               </b>
              </a>
              , you can check the quality of the burns your drive is producing
             </li>
            </ul>
           </td>
          </tr>

           <td class="contents" colspan="2" width="50%">
            <p>
             ImgBurn supports a wide range of image file formats - including BIN, CCD, CDI, CUE, DI, DVD, GI, IMG, ISO, MDS, NRG and PDI.
            </p>
            <p>
             It can burn Audio CD's from any file type supported via DirectShow / ACM - including AAC, APE, FLAC, M4A, MP3, MP4, MPC, OGG, PCM, WAV, WMA and WV.
            </p>
            <p>
             You can use it to build DVD Video discs (from a VIDEO_TS folder), HD DVD Video discs (from a HVDVD_TS folder) and Blu-ray Video discs (from a BDAV / BDMV folder) with ease.
            </p>
            <p>
             It supports
             <a href="https://www.unicode.org/" target="_blank" title="Click here to visit the Unicode website (www.unicode.org)">
              <b>
               Unicode
              </b>
             </a>
             folder/file names, so you shouldn't run in to any problems if you're using an international character set.
            </p>
            <p>
             ImgBurn supports all the
             <a href="https://www.microsoft.com/" target="_blank" title="Click here to visit the Microsoft website (www.microsoft.com)">
              <b>
               Microsoft Windows
              </b>
             </a>
             OS's - Windows 95, Windows 98, Windows Me, Windows NT4, Windows 2000, Windows XP, Windows 2003, Windows Vista, Windows 2008, Windows 7, Windows 8 and Windows 10 (including all the 64-bit versions). If you use
             <a href="https://www.winehq.com/" target="_blank" title="Click here to visit the WineHQ website (www.winehq.com)">
              <b>
               Wine
              </b>
             </a>
             , it should also run on
             <a href="https://www.linux.org/" target="_blank" title="Click here to visit the Linux website (www.linux.org)">
              <b>
               Linux
              </b>
             </a>
             and other x86-based Unixes.
            </p>
            <p>
             It's a very flexible application with several advanced features that are often lacking in other tools, especially when it comes to burning DVD Video discs. It supports all the latest drives without the need for updates (including booktype / bitsetting / advanced settings on many of the major ones - i.e. BenQ, LiteOn, LG, NEC, Optiarc, Pioneer, Plextor, Samsung, Sony).
            </p>
            <p>
             There is an
             <a href="index.php?act=screenshots#isowrite_queue" title="Click here to see a screenshot of this feature">
              image queue
             </a>
             system for when you're burning several images (which you can automatically share between multiple drives if you have more than one) and an easy-to-use
             <a href="index.php?act=screenshots#layer_break" title="Click here to see a screenshot of this feature">
              layer break selection
             </a>
             screen for double layer DVD Video jobs. The
             <a href="index.php?act=screenshots#automatic_write_speed" title="Click here to see a screenshot of this feature">
              Automatic Write Speed
             </a>
             feature allows you store your favourite burn speed settings on a per 'Media ID' basis, right down to a drive by drive level. Data captured during the burn (write speed, buffer levels etc) can be
             <a href="index.php?act=screenshots#ibg" title="Click here to see a screenshot of this feature">
              displayed / analysed
             </a>
             using
             <a href="http://www.dvdinfopro.com/" target="_blank" title="Click here to visit the DVDInfoPro website (www.dvdinfopro.com)">
              <b>
               DVDInfoPro
              </b>
             </a>
             .
            </p>
            <p>
             Whilst ImgBurn is designed to work perfectly straight out of the box, advanced users will appreciate just how configurable it is.
            </p>
            <p>
             Oh and let's not forget the best thing about it.... it's 100% FREE ;-)
            </p>
            <p>
             <b>
              Please do not use this software to make illegal copies of copyrighted discs.
             </b>
            </p>
