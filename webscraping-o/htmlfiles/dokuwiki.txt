        <h1 class="sectionedit1" id="dokuwiki">
         DokuWiki
        </h1>
        <div class="level1">
         <p>
          DokuWiki is a simple to use and highly versatile Open Source
          <a class="interwiki iw_wp" href="https://en.wikipedia.org/wiki/wiki" title="https://en.wikipedia.org/wiki/wiki">
           wiki
          </a>
          software that doesn't require a database. It is loved by users for its clean and readable
          <a class="wikilink1" data-wiki-id="wiki:syntax" href="/wiki:syntax" title="wiki:syntax">
           syntax
          </a>
          . The ease of maintenance, backup and integration makes it an administrator's favorite. Built in
          <a class="wikilink1" data-wiki-id="acl" href="/acl" title="acl">
           access controls
          </a>
          and
          <a class="wikilink1" data-wiki-id="auth" href="/auth" title="auth">
           authentication connectors
          </a>
          make DokuWiki especially useful in the enterprise context and the large number of
          <a class="wikilink1" data-wiki-id="plugins" href="/plugins" title="plugins">
           plugins
          </a>
          contributed by its vibrant community allow for a broad range of use cases beyond a traditional wiki.
         </p>
         <p>
          Get started:
          <a class="media" href="http://download.dokuwiki.org" title="http://download.dokuwiki.org">
           <img alt="Download the latest version" class="media" src="/_media/marketing:download-big.png" title="Download the latest version"/>
          </a>
         </p>
         <hr/>
         <div class="dokuteaser">
          <h3 class="sectionedit4" id="why_dokuwiki">
           Why DokuWiki?
          </h3>
          <p>
           <img alt="" class="mediaright" src="/_media/wiki:dokuwiki-128.png?w=80&amp;tok=a6886f" width="80"/>
           DokuWiki is a popular choice when choosing a Wiki software and has many advantages over similar software.
          </p>
          <ul>
           <li class="level1">
            <div class="li">
             Easy to install and use
            </div>
           </li>
           <li class="level1">
            <div class="li">
             Low system requirements
            </div>
           </li>
           <li class="level1">
            <div class="li">
             Built-in Access Control Lists
            </div>
           </li>
           <li class="level1">
            <div class="li">
             Large variety of extensions
            </div>
           </li>
           <li class="level1">
            <div class="li">
             Over 50 languages supported
            </div>
           </li>
           <li class="level1">
            <div class="li">
             Device independent
            </div>
           </li>
           <li class="level1">
            <div class="li">
             Open Source
            </div>
           </li>
          </ul>
          <p>
           <a class="wikilink1" data-wiki-id="features" href="/features" title="features">
            Learn more about DokuWiki features
           </a>
          </p>
         </div>
