                                                 <h1 class="sectionedit1 page-header pb-3 mb-4 mt-5" id="welcome_to_the_openwrt_project">
                                                  Welcome to the OpenWrt Project
                                                 </h1>
                                                 <div class="level1">
                                                  <p>
                                                   The OpenWrt Project is a Linux operating system targeting embedded devices. Instead of trying to create a single, static firmware, OpenWrt provides a fully writable filesystem with package management. This frees you from the application selection and configuration provided by the vendor and allows you to customize the device through the use of packages to suit any application. For developers, OpenWrt is the framework to build an application without having to build a complete firmware around it; for users this means the ability for full customization, to use the device in ways never envisioned.
                                                  </p>
                                                  <p>
                                                   See the
                                                   <a class="wikilink1" data-wiki-id="toh:start" href="/toh/start" title="toh:start">
                                                    Table of Hardware
                                                   </a>
                                                   for supported devices. For more information about OpenWrt project organization, see the
                                                   <a class="wikilink1" data-wiki-id="about" href="/about" title="about">
                                                    About OpenWrt
                                                   </a>
                                                   pages.
                                                  </p>
                                                 </div>
                                                 <h2 class="sectionedit2 page-header pb-3 mb-4 mt-5" id="openwrt_joins_software_freedom_conservancy_sfc">
                                                  OpenWrt Joins Software Freedom Conservancy (SFC)
                                                 </h2>
                                                 <div class="level2">
                                                 </div>
                                                 <div class="plugin_include_content plugin_include__openwrt_joins_conservancy">
                                                  <div class="level2">
                                                   <p>
                                                    SFC has welcomed us into their organization
                                                    <sup>
                                                     <a class="fn_top" href="#fn__1" id="fnt__1">
                                                      1)
                                                     </a>
                                                    </sup>
                                                    :
                                                   </p>
                                                   <p>
                                                    <em>
                                                     September 10, 2020

OpenWrt — building on their sixteen years of success as the most popular Free and Open Source (
                                                     <abbr title="Free &amp; Open-Source Software">
                                                      FOSS
                                                     </abbr>
                                                     ) wireless router project — today joins
                                                     <a class="urlextern" href="https://sfconservancy.org/" rel="ugc nofollow" title="https://sfconservancy.org/">
                                                      Conservancy
                                                     </a>
                                                     as a member project.
                                                     <abbr title="Free &amp; Open-Source Software">
                                                      FOSS
                                                     </abbr>
                                                     wireless routers assure software freedom for all Internet users. Conservancy will help OpenWrt continue to thrive and grow as its new fiscal sponsor.

OpenWrt occupies a special place in the history of software freedom. OpenWrt's creation and launch shows that
                                                     <a class="urlextern" href="https://sfconservancy.org/copyleft-compliance/" rel="ugc nofollow" title="https://sfconservancy.org/copyleft-compliance/">
                                                      GPL enforcement
                                                     </a>
                                                     works and advances software freedom. In 2004, when Linksys released the firmware code for the WRT54G router series, coders and tinkerers regained control over their own routers, and
                                                     <a class="wikilink1" data-wiki-id="about:history" href="/about/history" title="about:history">
                                                      launched OpenWrt based on the sources liberated from GPL enforcement
                                                     </a>
                                                     . Today, OpenWrt leverages software freedom protected by the
                                                     <abbr title="GNU General Public License">
                                                      GPL
                                                     </abbr>
                                                     to share that freedom and control with everyone who uses wireless routers to connect to the Internet.
                                                    </em>
                                                   </p>
                                                   <p class="include_readmore">
                                                    <a class="wikilink1" data-wiki-id="openwrt_joins_conservancy" href="/openwrt_joins_conservancy" title="openwrt_joins_conservancy">
                                                     → Read more...
                                                    </a>
                                                   </p>
                                                  </div>
                                                 </div>
                                                 <div class="level2">
                                                 </div>
                                                 <h2 class="sectionedit5 page-header pb-3 mb-4 mt-5" id="current_stable_seriesopenwrt_1907">
                                                  Current stable series: OpenWrt 19.07
                                                 </h2>
                                                 <div class="level2">
                                                 </div>
                                                 <div class="plugin_include_content plugin_include__releases:19.07:start">
                                                  <div class="level2">
                                                   <p>
                                                    The OpenWrt Community is proud to present the OpenWrt 19.07 stable version series. It is the successor of the previous 18.06 stable major release.
                                                   </p>
                                                   <p>
                                                    The OpenWrt 19.07 series focuses on bringing all supported targets to Linux kernel version 4.14 and introducing initial device tree based
                                                    <a class="wikilink1" data-wiki-id="docs:techref:targets:ath79" href="/docs/techref/targets/ath79" title="docs:techref:targets:ath79">
                                                     ath79
                                                    </a>
                                                    support.
                                                   </p>
                                                   <p>
                                                    <strong>
                                                     Current Stable Release - OpenWrt 19.07.7
                                                    </strong>
                                                   </p>
                                                   <p>
                                                    The current stable version series of OpenWrt is 19.07, with
                                                    <strong>
                                                     v19.07.7
                                                    </strong>
                                                    being the latest release of the series. It was released on 18 February 2021.
                                                   </p>
                                                   <ul class="fix-media-list-overlap">
                                                    <li class="level1">
                                                     <div class="li">
                                                      <strong>
                                                       <a class="wikilink1" data-wiki-id="releases:19.07:notes-19.07.7" href="/releases/19.07/notes-19.07.7" title="releases:19.07:notes-19.07.7">
                                                        Release Notes
                                                       </a>
                                                      </strong>
                                                     </div>

                                                 <h3 class="sectionedit8 page-header pb-3 mb-4 mt-5" id="old_stable_seriesopenwrt_1806">
                                                  Old stable series: OpenWrt 18.06
                                                 </h3>
                                                 <div class="level3">
                                                 </div>
                                                 <div class="plugin_include_content plugin_include__releases:18.06:start">
                                                  <div class="level3">
                                                  </div>
                                                  <div class="plugin_include_content plugin_include__meta:infobox:outdated_release" id="plugin_include__meta__infobox__outdated_release">
                                                   <div class="level4">
                                                    <div class="wrap_outdated wrap_center plugin_wrap" style="width: 70%;">
                                                     <p>
                                                      <strong>
                                                       Outdated release!
                                                      </strong>
                                                      <br/>
                                                      This OpenWrt release is no longer receiving updates. You should switch to a newer release.
                                                     </p>
                                                    </div>
                                                   </div>
                                                  </div>
                                                  <div class="level4">
                                                   <p>
                                                    The OpenWrt Community is proud to present the OpenWrt 18.06 stable version series. It is the first stable version after the OpenWrt/LEDE project merger and the successor to the previous stable LEDE 17.01 and OpenWrt 15.05 major releases.
                                                   </p>
                                                   <p>
                                                    The OpenWrt 18.06 series focuses on modernizing many parts of the system, on backporting network offload support for eligible targets and on laying the groundwork for regular future release updates.
                                                   </p>
                                                   <p>
                                                    <strong>
                                                     Current old-stable release - OpenWrt 18.06.9
                                                    </strong>
                                                   </p>
                                                   <p>
                                                    The current old-stable version series of OpenWrt is 18.06, with
                                                    <strong>
                                                     v18.06.9
                                                    </strong>
                                                    being the last service release of the series. It was released on 9 December 2020. No new releases in this series are expected.
                                                   </p>
                                                   <ul class="fix-media-list-overlap">
                                                    <li class="level1">

                                                 <h2 class="sectionedit15 page-header pb-3 mb-4 mt-5" id="why_use_openwrt">
                                                  Why use OpenWrt?
                                                 </h2>
                                                 <div class="level2">
                                                  <p>
                                                   People install OpenWrt because they believe it works better than the stock firmware from their vendor. They find it is more stable, offers more features, is more secure and has better support.
                                                  </p>
                                                  <ul class="fix-media-list-overlap">
                                                   <li class="level1">
                                                    <div class="li">
                                                     <strong>
                                                      Extensibility:
                                                     </strong>
                                                     OpenWrt provides many capabilities found only in high-end devices. Its 3000+ application packages are standardized, so you can easily replicate the same setup on any supported device, including two (or even five) year old routers.
                                                     <a class="wikilink1" data-wiki-id="reasons_to_use_openwrt" href="/reasons_to_use_openwrt#extensibility" title="reasons_to_use_openwrt">
                                                      More...
                                                     </a>
                                                    </div>
                                                   </li>
                                                  </ul>
                                                  <ul class="fix-media-list-overlap">
                                                   <li class="level1">
                                                    <div class="li">
                                                     <strong>
                                                      Security:
                                                     </strong>
                                                     OpenWrt's standard installation is secure by default, with Wi-Fi disabled, no poor passwords or backdoors. OpenWrt's software components are kept up-to-date, so
                                                     <a class="wikilink1" data-wiki-id="advisory:start" href="/advisory/start" title="advisory:start">
                                                      vulnerabilities get closed shortly after they are discovered
                                                     </a>
                                                     .
                                                     <a class="wikilink1" data-wiki-id="reasons_to_use_openwrt" href="/reasons_to_use_openwrt#security" title="reasons_to_use_openwrt">
                                                      More...
                                                     </a>
                                                    </div>
                                                   </li>
                                                  </ul>
                                                  <ul class="fix-media-list-overlap">
                                                   <li class="level1">
                                                    <div class="li">
                                                     <strong>
                                                      Performance and Stability:
                                                     </strong>
                                                     OpenWrt firmware is made of standardized modules used in all supported devices. This means each module will likely receive more testing and bug fixing than stock firmware which can be tweaked for each product line and never touched again.
                                                     <a class="wikilink1" data-wiki-id="reasons_to_use_openwrt" href="/reasons_to_use_openwrt#performance_stability" title="reasons_to_use_openwrt">
                                                      More...
                                                     </a>
                                                    </div>
                                                   </li>
                                                  </ul>
                                                  <ul class="fix-media-list-overlap">
                                                   <li class="level1">
                                                    <div class="li">
                                                     <strong>
                                                      Strong Community Support:
                                                     </strong>
                                                     OpenWrt team members are regular participants on the
                                                     <a class="urlextern" href="https://forum.openwrt.org" rel="ugc nofollow" title="https://forum.openwrt.org">
                                                      OpenWrt Forum
                                                     </a>
                                                     ,
                                                     <a class="urlextern" href="https://lists.openwrt.org/mailman/listinfo/openwrt-devel" rel="ugc nofollow" title="https://lists.openwrt.org/mailman/listinfo/openwrt-devel">
                                                      OpenWrt Developer
                                                     </a>
                                                     and
                                                     <a class="urlextern" href="https://lists.openwrt.org/mailman/listinfo/openwrt-adm" rel="ugc nofollow" title="https://lists.openwrt.org/mailman/listinfo/openwrt-adm">
                                                      OpenWrt Admin
                                                     </a>
                                                     mailing lists, and
                                                     <a class="wikilink1" data-wiki-id="contact" href="/contact#irc_channels" title="contact">
                                                      LEDE's IRC channels.
                                                     </a>
                                                     You can interact directly with developers, volunteers managing the software modules and with other long-time OpenWrt users, drastically increasing the chances you will solve the issue at hand.
                                                     <a class="wikilink1" data-wiki-id="contact" href="/contact" title="contact">
                                                      More...
                                                     </a>
                                                    </div>
                                                   </li>
                                                  </ul>
                                                  <ul class="fix-media-list-overlap">
                                                   <li class="level1">
                                                    <div class="li">
                                                     <strong>
                                                      Research:
                                                     </strong>
                                                     Many teams use OpenWrt as a platform for their research into network performance. This means that the improvements of their successful experiments will be available in OpenWrt first, well before it gets incorporated into mainline, vendor firmware.
                                                     <a class="wikilink1" data-wiki-id="reasons_to_use_openwrt" href="/reasons_to_use_openwrt#research_platform" title="reasons_to_use_openwrt">
                                                      More...
                                                     </a>
                                                    </div>
                                                   </li>
                                                  </ul>
                                                  <ul class="fix-media-list-overlap">
                                                   <li class="level1">
                                                    <div class="li">
                                                     <strong>
                                                      Open Source/No additional cost:
                                                     </strong>
                                                     OpenWrt is provided without any monetary cost. It has been entirely created by a team of volunteers: developers and maintainers, individuals and companies. If you enjoy using OpenWrt, consider contributing some effort to
                                                     <a class="wikilink1" href="#openwrt_wants_you" title="start ↵">
                                                      help us improve it for others!
                                                     </a>
                                                     All of the above is possible because OpenWrt is part of the Open Source community, and powered by Linux kernel.
                                                     <a class="urlextern" href="https://git.openwrt.org" rel="ugc nofollow" title="https://git.openwrt.org">
                                                      Get the source code...
                                                     </a>
                                                    </div>
                                                   </li>
                                                  </ul>
                                                  <ul class="fix-media-list-overlap">
                                                   <li class="level1">
                                                    <div class="li">
                                                     <strong>
                                                      And more…
                                                     </strong>
                                                    </div>
                                                   </li>
                                                  </ul>
                                                  <div class="wrap_right plugin_wrap">
                                                   <p>
                                                    <a class="wikilink1" data-wiki-id="reasons_to_use_openwrt" href="/reasons_to_use_openwrt#configuration" title="reasons_to_use_openwrt">
                                                     → Read more reasons to use OpenWrt...
                                                    </a>
                                                   </p>
                                                  </div>

                                                 <h2 class="sectionedit18 page-header pb-3 mb-4 mt-5" id="announcing_the_openwrtlede_merge">
                                                  Announcing the OpenWrt/LEDE merge
                                                 </h2>
                                                 <div class="level2">
                                                 </div>
                                                 <div class="plugin_include_content plugin_include__about">
                                                  <div class="level2">
                                                   <p>
                                                    As of January 2018, both the OpenWrt and LEDE projects agreed to re-merge back under the OpenWrt name.
                                                   </p>
                                                   <p>
                                                    The new, unified OpenWrt project is governed under the
                                                    <a class="wikilink1" data-wiki-id="rules" href="/rules" title="rules">
                                                     rules established by the former LEDE project
                                                    </a>
                                                    . Active members of both the former LEDE and OpenWrt projects contribute to the unified OpenWrt.
                                                   </p>
                                                   <p class="include_readmore">
                                                    <a class="wikilink1" data-wiki-id="about" href="/about" title="about">
                                                     → Read more...
                                                    </a>
                                                   </p>
                                                  </div>

                                                 <h2 class="sectionedit21 page-header pb-3 mb-4 mt-5" id="openwrt_wants_you">
                                                  OpenWrt wants you!
                                                 </h2>
                                                 <div class="level2">
                                                  <p>
                                                   Like any open source project, OpenWrt thrives on the efforts of its users and developers.
                                                  </p>
                                                  <ul class="fix-media-list-overlap">
                                                   <li class="level1">
                                                    <div class="li">
                                                     If you want to develop the software, please refer to our
                                                     <a class="wikilink1" data-wiki-id="docs:guide-developer:source-code:start" href="/docs/guide-developer/source-code/start" title="docs:guide-developer:source-code:start">
                                                      Developer Guide
                                                     </a>
                                                     to learn how to get the source code, build it, and contribute your changes back to the project.
                                                    </div>
                                                   </li>
                                                  </ul>
                                                  <ul class="fix-media-list-overlap">
                                                   <li class="level1">
                                                    <div class="li">
                                                     If you're not a developer, you can still help. The documentation can always be improved (even if it's to verify that the instructions match your experience), or you can help other community members with questions. Don’t hesitate to
                                                     <a class="wikilink1" data-wiki-id="register_yourself_in_the_wiki" href="/register_yourself_in_the_wiki" title="register_yourself_in_the_wiki">
                                                      Register yourself in the wiki
                                                     </a>
                                                     , or join our
                                                     <a class="urlextern" href="https://lists.openwrt.org/mailman/listinfo/openwrt-devel" rel="ugc nofollow" title="https://lists.openwrt.org/mailman/listinfo/openwrt-devel">
                                                      mailing list
                                                     </a>
                                                     and
                                                     <a class="wikilink1" data-wiki-id="contact" href="/contact#irc_channels" title="contact">
                                                      IRC channels
                                                     </a>
                                                     to get in touch.
                                                    </div>
                                                   </li>
                                                  </ul>
